# Documentation Prompt

Используй этот промпт для документирования кода.

---

Создай документацию для следующего кода:

**Код для документирования:**
```[язык]
[вставь код]
```

**Требования к документации:**

1. **Docstrings/Comments:**
   - Опиши что делает функция/класс/модуль
   - Опиши параметры (типы, назначение)
   - Опиши возвращаемое значение
   - Укажи возможные исключения
   - Добавь примеры использования если уместно

2. **Стиль:**
   - Следуй conventions из `.ai/conventions.md`
   - Используй [стиль docstring: Google / NumPy / reStructuredText]
   - Будь кратким но информативным

3. **Дополнительно:**
   - [ ] Опиши сложную бизнес-логику
   - [ ] Укажи edge cases
   - [ ] Добавь TODO если что-то нужно улучшить
   - [ ] Опиши зависимости от других модулей

**Пример формата:**
```[язык]
def example(param1: str, param2: int) -> bool:
    """
    Краткое описание что делает функция.
    
    Подробное описание если нужно объяснить сложную логику
    или бизнес-правила.
    
    Args:
        param1: Описание первого параметра
        param2: Описание второго параметра
    
    Returns:
        Описание возвращаемого значения
    
    Raises:
        ValueError: Когда происходит ошибка валидации
        
    Example:
        >>> example("test", 42)
        True
    """
    pass
```

---

**Результат:**
Предоставь полностью задокументированный код с объяснениями.
