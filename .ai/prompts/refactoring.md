# Refactoring Prompt

Используй этот промпт когда нужно отрефакторить код.

---

Отрефактори следующий код с учётом:

1. **Conventions:** Следуй соглашениям из `.ai/conventions.md`
2. **Architecture:** Соблюдай архитектуру из `.ai/architecture.md`
3. **Principles:**
   - SOLID принципы
   - DRY (Don't Repeat Yourself)
   - KISS (Keep It Simple, Stupid)
   - Читаемость кода

4. **Requirements:**
   - Сохрани существующую функциональность
   - Все существующие тесты должны проходить
   - Добавь комментарии где необходимо
   - Улучши именование переменных/функций если нужно

5. **Focus areas:**
   - [ ] Удалить дублирование кода
   - [ ] Упростить сложную логику
   - [ ] Разбить большие функции
   - [ ] Улучшить читаемость
   - [ ] Оптимизировать производительность (если критично)

**Код для рефакторинга:**
```[язык]
[вставь код здесь]
```

**Контекст (опционально):**
[Опиши контекст: зачем рефакторинг, какие проблемы решаем]

---

**После рефакторинга предоставь:**
1. Отрефакторенный код
2. Краткое объяснение основных изменений
3. Что улучшилось
