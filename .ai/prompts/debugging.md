# Debugging Prompt

Используй этот промпт для отладки проблем в коде.

---

Помоги отладить следующую проблему:

**Проблема:**
[Опиши проблему: что не работает, какое ожидаемое поведение]

**Код с проблемой:**
```[язык]
[вставь проблемный код]
```

**Ошибка / Лог:**
```
[вставь ошибку или лог если есть]
```

**Что уже пробовал:**
- [Попытка 1]
- [Попытка 2]

**Контекст:**
- **Окружение:** [dev/staging/production]
- **Версии:** [версии зависимостей если релевантно]
- **Когда началось:** [если известно]

**Checklist для анализа:**
- [ ] Проверь логи приложения
- [ ] Проверь переменные окружения
- [ ] Проверь состояние БД
- [ ] Проверь права доступа / permissions
- [ ] Проверь версии зависимостей
- [ ] Воспроизводится ли на других окружениях?

---

**Нужна помощь с:**
1. Определением причины проблемы
2. Предложением решения
3. Объяснением почему это происходит
4. Способами предотвращения в будущем
